# Bank Statement Analysis

## 📌 Overview
This project analyzes raw bank statement data to identify spending patterns, categorize transactions, and generate financial insights.  
It demonstrates my ability to work with real-world financial data, automate reporting, and present clear results in a professional format.

## 🎯 Objectives
- Automate the process of cleaning and organizing bank statement data.
- Categorize transactions (e.g., food, transport, utilities, savings).
- Generate summary reports and visualizations of spending habits.
- Provide actionable insights for budgeting and financial planning.

## 🛠 Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib/Seaborn)
- **Excel** (for data validation and presentation)
- **Jupyter Notebook** (for analysis and documentation)
- **GitHub** (for version control and portfolio showcase)

## 📂 Features
- Transaction categorization by type and merchant.
- Monthly and yearly spending summaries.
- Visualizations (pie charts, bar graphs, trend lines).
- Exportable reports for easy sharing.
